Humidor for Microbial Community Analysis using CIGAR strings

The Humidor microbial community classifier uses convolutional neural networks in order to classify large amounts of 16S gene sequence data. Pre-existing classifiers had a lot of trouble scaling with the huge increase in data brought on be next-gen sequencing technologies. Thus, dimensionality reduction was necessary, and this consisted of clustering all the data into equivalence classes. However, generating these classes proved to be troublesome. The most widely used technique clustered based on genetic distance between two sequences; however, a single species, like E. Coli, can range from 4 million base pairs to 10 billion base pairs - very unlikely to be genetically 'close'. Thus, with the advanced in deep learning, we proposed a new technique that would not require pre-clustering. Moreover, we found that this technique performed similarly to the current RDP classifer, but did not require the dimensionality reduction step when scaled. See the full paper below.
