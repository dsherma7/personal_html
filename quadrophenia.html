Quadrophenia Dosage Calling

The Quadrophenia dosage calling algorithm is the last step of the laboratory management information system, LIMS, that I developed for our lab processes. This algorithm uses a mix of semi and fully supervised learning models to train on data that consists of only 10% labeled samples. Moreover, a bootstrapping method is used to determine the likelihood of correct calls. This model also works on presence absence style experiments that focus only on detecting the presence of a gene rather than how many copies of it exist. These types of experiments can be run faster and are used only when presence firewalls a sample. Access the R package repository here. For discretion, this repository is private, so email dsherman@landolakes.com to request access.
