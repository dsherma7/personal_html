Portfolio Manager

At Forage Genetics I lead the development of a Portfolio Manager decision support tool. This tool's main purpose was to facilitate the end-of-year product advancement decisions; which often were riddled with incosistencies and poor reporting. To do this, I researched some options in data visualization and cloud computing, and I decided to leverage Google Cloud and Tableau. Since the majority of this tool is proprietary, I am only displaying the tutorials I made to ensure full adoption. However, I would be happy to go over any piece if you have questions.

The first step in the process is to use the Performance Summary to analyze all of the data throughout the year. This data consists of yield, Quality, and pest resistance values for each of the trials throughout each region. The performance summary is a Tableau workbook that allows each user to filter down to specific trials and study the data in each trial. Moreover, this workbook can be used to mark experimentals for further review in the Nomination Dashboard.

The Nomination Dashboard allows for the user to review the samples marked for review in the Performance Summary. When the best sample is determined, it can be Nominated. All the users Nominations can then be reviewed simultaneously in the All Nominations page. At the yearly product advancement meeting, the agglomerated nominations can be hand-selectd by the advancement team and Our Advancements will display these decisions.

The Nomination Overview Tableau workbook is used to look deeper into the samples that are in the Nomination Dashboard. This allows to look at much more detail using visuals, where the Nomination Dashboard only provides a summary table. Moreover, users can mark these samples as 'dropped' if they discover a feature of them that eliminates their ability to be advanced.
