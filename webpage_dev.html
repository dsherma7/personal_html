How to Build a website with Bootstrap

In this page I go over some of the basics for developing a quick but effective website. We will start by downloading a readily available Bootstrap CSS template, learn what the HTML is saying, and how to copy this page to an apache server for hosting. This intro is designed for people with beginner to intermediate level programming skills.

What is in the HTML?

Brief descriptions of what different parts of the Bootsrap html are doing

<head>
The <head> tag is used at the top of a page and represents any meta data for your web page. This includes information about the author or title, .js scripts, and links to your .css files. See more about the <head> tag here.
<b> <i> <em> <small> <a>
These tags are for fomatting your text. The <b> makes it bold, <i> makes it italicized, <em> emphasizes the text (i.e bold + italics), <small> makes it smaller, and <a> creates a hyperlink. Learn more about formatting tags here.
<div>
The <div> tag is the primary tag for formatting sections of your web page. This tag can divide up the space in your page based on the class= you specify. Consider the full length of the screen as 12 units, then col-lg-x will take x units of this length. See more about <div> tag here.
<ul>
The <ul> tag creates an unordered list. Once this tag is opened, you can then use <li> to list consecutive list items. Moreover, you can use <ol> to create an ordered list. Learn more about lists here.
CSS
The style.css document is how to format most of the page. These options can be applied on a <div> tag as <div class="nav-menu">. Check here for more.
Extend CSS
The CSS classes you create can adjust any tag inside of their scope. Here we see that the <ul> tag is adjusted by the nav-menu. With these extensions, you can build sections in your html code and format each tag differently. With these, you can even control special effects like hovering. See more advanced CSS tricks here.

Host on a Server

Finally, we need to load this onto the UC Davis Math server. Since the Math department has set up an apache server, we can do this by simply moving all our files to our math home directory. This can be done with a flash drive, or by using scp. For grad students, the server to use is either Fuzzy or Logic, and the host name is math.ucdavis.edu. Thus the scp command to use is 
scp website.zip [username]@fuzzy.math.ucdavis.edu:
where website.zip is the name of a compressed version of your website's directory and [username] is your username for the Math department (not UC Davis Kerboros). Once you've done this you can log into the math server with 
ssh [username]@fuzzy.math.ucdavis.edu
This takes you to your home directory on the fuzzy math server. To host your website, you need to a directory called public_html. The best way to do this is to unzip your directory and rename it to public_html with 
unzip website.zip
mv website/ public_html/
It may take a few minutes to register, but now you're done! You can access your website by going to www.math.ucdavis.edu/~[username]. Have fun building your page and contact me with any questions at desherman@ucdavis.edu!
